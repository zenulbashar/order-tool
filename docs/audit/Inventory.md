# Product Inventory — Prompt2Eat (order-tool)

_Generated by automated discovery of the App Router tree, `lib/` services, the
Drizzle schema, and shared UI primitives. Not a hand-maintained list._

Stack: **Next.js 16.2.9** (App Router, React 19.2), **Tailwind v4**, **Drizzle
ORM** (Postgres/Neon), **NextAuth v5 (beta)**, **Stripe** + **Square**, S3/R2
uploads, Upstash rate-limiting, Anthropic SDK (AI concierge + support).

Scale: 195 `.tsx` components · 105 client components · 60 pages · 13 route
handlers · 36 server-action modules · 43 DB tables · 57 migrations.
